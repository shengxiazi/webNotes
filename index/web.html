1.正则表达式(正则对象)

  • 01.正则表达式的声明和使用

    ①构造函数

    var regEx = new RegExp(/\d/);

    ②直接量

    var regEx = /\w/;

    方法：regEx.test("a");

  • 02.预定义类和转义符

    . [^\n\r] 除了换行和回车之外的任意字符

    \d [0-9] 数字字符digit

    \D [^0-9] 非数字字符

    \w [a-zA-Z0-9_] 单词字符(所有的字母数字和_) word

    \W [^a-zA-Z0-9_] 非单词字符

    \s [\f\r\n\t\v] 不可见字符 space

    \S [^\f\r\n\t\v] 可见字符

    \f 表示换页 form feed

    \t 表示水平制表符 table

    \v 表示垂直制表符 vertical table

  • 03.字符类

    简单类[abc] 表示该位置可以出现的字符

    负向类[^abc] 表示该位置不可以出现的字符

    范围类[a-e] 表示该位置可以出现的字符的范围

    组合类[a-xA-E0-9] 范围类的组合

  • 04.边界 量词 括号

    ^ 会匹配行或者字符串的起始位置

    ^只有在[]内才表示非 在外边表示开始

    $ 会匹配行或字符串的结尾位置

    ^$在一起 表示必须是这个（精确匹配）

    "*" 重复零次或更多 x>=0

    "+" 重复一次或更多次 x>=1

    "?" 重复零次或一次 x=(0||1)

    {n} n次 x=n

    {n,} 重复n次或更多 x>=n

    {n,m} 重复出现的次数比n多但比m少 n<=x<=m

    ()表示一组

    []表示一个字符的位置

    {}表示次数

    匹配国内固定电话号码：/^0\d{2,3}-\d{7,8}$/

    匹配姓名：/^[\u4e00-\u9fa5]{2,}$/

    匹配腾讯QQ号：/^[1-9]\d{4,10}$/

    匹配手机号：/^(13[0-9]|14[57]|15[0-9]|18[0-9])\d{8}$/

    匹配邮箱：/^\w+([+-.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*$/

  • 05.封装trim方法（检测用户输入）

    function trim(str) {

    return str.replace(/^\s+/,"").replace(/\s+$/,"");

    };

2.事件对象

  • 事件对象简介

    在触发事件时，会产生一个事件对象event，这个对象中包含着与事件有关的信息。

    标准浏览器支持 传入参数
    ie 678 支持 window.event

    事件对象的属性: pageX  pageY  clientX  clientY  target  type

  • 放大镜特效

    鼠标在盒子中的位置：水平位置 pageX-offsetLeft 垂直位置 pageY-offsetTop

    比例rate：B整体/A整体 = B部分/A部分

    可拖拽特效

    盒子的位置要进行调整，当鼠标按下时，记录鼠标在盒子中的位置，鼠标移动时计算盒子位置的时候减去。

  • 注册事件的两种方式

    添加事件监听器(管理事件队列)element.addEventListener(type,fn,false);

    移除事件 element.removeEventListener(type, fn, false);

  • 事件的三个阶段

    1.捕获阶段

    2.目标阶段 执行当前对象的事件处理程序

    3.冒泡阶段（阻止事件冒泡）

  • 事件

    鼠标事件:

    onmousedown、onmouseup、
    onmousemove、onmouseover、onmouseout、onclick、ondblclick

    onfocus、onblur

    onresize、onscroll

    键盘事件：

    onkeydown、onkeyup、onkeypress onkeydown onkeyup 输出的是键盘码
    onkeypress输出的是ASCII码表

    onload、unload

3.jQuery（js库）

  • 01.jQuery简介

    遇到的问题：
    1. window.onload 事件只能出 现一次,如果出现多次，后面的事件就把前面的事件给覆盖掉
    2. 代码的容错性差
    3. 实现简单功能，代码很繁琐
    4. 浏览器兼容性问题

    jQuery是js的一个库，封装了我们开发过程中常用的一些功能和方法API

    在使用jQuery的API时，都是方法调用，也就是说要加小括号()
    ，小括号里面是相应的参数，参数不同，功能不同。

    DOM树加载完成后，就可以操作DOM了，不用等到所有的外部资源都加载完成。

    （补充）js入口函数：window.onload = function() {};
    只能出现一次，出现多次会存在事件覆盖的问题。

    功能实现代码（事件处理）jQuery隐式迭代在方法的内部进行循环遍历
